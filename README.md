# password_advisor
AI-powered Password Strength Advisor that estimates password cracking time, visualizes risk, and suggests stronger passwords in real-time.

## Features
- Calculates password **entropy** and **time-to-crack**
- Suggests improvements such as adding uppercase letters, numbers, symbols, or increasing length
- Visualizes password strength as Weak ⚠️, Moderate ⚠️, or Strong ✅

## Tech Stack
- Python
- Streamlit
